# 硬件参考

本页只记录当前项目中已经确认的硬件型号/系列，不展开参数。

## 已确认硬件

| 类别 | 型号 / 系列 | 说明 |
| --- | --- | --- |
| 运动控制板卡 | 博派科技 `COM_GAS_N` | 当前软件板卡接口与动态库调用都基于 `COM_GAS_N` |
| 电机 | `AIMotor` 低压系列一体化伺服电机 | 当前参考资料能确认到“低压系列”，未见更细的具体料号 |
| 光机 | 闻亭泰科 `4710` | 当前光机控制命令模板来自 `pro4710.ini` |

## 当前 I/O 映射

| 信号 | 默认 DI | 说明 |
| --- | --- | --- |
| 孔位激光对射 | `DI0` | Z 到位确认使用，默认 `z_laser_active_low=true` |
| 旋转台限位开关 | `DI1` | 快速回零与精确回零使用，默认 `fine_home_active_low=false` |
| 急停 | `DI2` | 触发后进入安全停机 |
| 伺服报警 | `DI3` | 触发后进入安全停机 |
